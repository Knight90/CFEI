- CFEI得名自中国金融经济信息库的英文缩写

### 原银监会管理部分

 [原银监会金融许可证查询](http://xukezheng.cbrc.gov.cn/ilicence/licence/licenceQuery.jsp)

- 可以查询银行、信托、资产、租赁、消费金融等类别许可证信息
